Configuring Virtual Memory for XML NOM Documents

In a Windows computer, click Start > Programs > Process Platform > Instance Name > Tools > Management Console.
 The Management Console window appears.
On the Management Console window, click Platform Properties. The Platform Properties dialog box appears.
In the list of properties, look for bus.xml.vm.maxsize property and assign the desired memory value to it. 
If it is not available, create the property and assign a value in MB to it.
Click  to save. The new property is added to Platform Properties.
Restart Process Platform.
Virtual memory for XML NOM documents is configured.
